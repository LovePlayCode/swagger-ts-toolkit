# 🎯 高级用法

本目录包含 swagger-ts-toolkit 的高级使用技巧和最佳实践。

## 📂 文件说明

- **error-handling.js** - 全面的错误处理策略和最佳实践
- **development-workflow.js** - 开发工作流程优化和最佳实践
- **real-world-usage.ts** - 真实项目中的使用案例

## 💡 主要内容

### 错误处理
- HTTP 错误捕获和处理
- 自定义错误转换
- 重试机制

### 开发工作流
- 代码生成自动化
- 类型检查和验证
- 开发环境配置

### 真实项目案例
- 完整的项目集成示例
- 性能优化技巧
- 生产环境部署

## 🚀 快速开始

```bash
# 错误处理示例
node error-handling.js

# 开发工作流示例
node development-workflow.js

# 真实项目示例
npx ts-node real-world-usage.ts
```
