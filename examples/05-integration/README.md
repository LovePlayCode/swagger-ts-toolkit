# 🔗 项目集成示例

本目录展示如何在实际项目中集成 swagger-ts-toolkit。

## 📂 目录说明

### microservices/
微服务架构的集成方案，包含：
- **setup.js** - 微服务环境设置和多服务配置

## 🎯 集成场景

### 微服务架构
- 多个独立服务的 API 管理
- 统一的类型定义和 API 接口
- 服务间通信优化

## 🚀 快速开始

```bash
# 微服务设置
node microservices/setup.js
```
